# FORNIRO - a project for the 2024 of the AMTIS summer academy
**Forniro** is a webapp for buying products through web platform.

## Technologies used:
- Frontend:
  - ThailWind CSS 
  - React
- Backend:
  - NestJS
  - MongoDB via Mongoose
  - Algolia Search DB
  - TypeScript
